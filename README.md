# ISL to Speech System

This project converts Indian Sign Language (ISL) gestures into speech.

## Tech Stack
- OpenCV (Video Capture)
- MediaPipe (Hand Landmarks)
- PyTorch (LSTM + Transformer)
- Google Gemini API (NLP)
- pyttsx3 (Text to Speech)

## Features
- Real-time gesture detection
- Sentence formation
- Speech output
